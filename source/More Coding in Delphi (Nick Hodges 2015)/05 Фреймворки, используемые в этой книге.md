Эта книга о написании кода на Delphi. В результате в ней используется ряд фреймворков с открытым исходным кодом для упрощения методов программирования, которые обсуждаются здесь. Я ссылаюсь на них здесь, в начале книги, чтобы вы могли получить код и настроить его перед началом работы.

Все описанные ниже фреймворки доступны через систему управления версиями. 
Я рекомендую вам извлечь или клонировать исходный код в соответствии с инструкциями в репозитории исходных текстов. Таким образом, вы сможете очень легко быть в курсе последних изменений и исправлений. Если у вас есть вопросы о том, как настроить и использовать Subversion, Git или Mercurial, пожалуйста, обратитесь к Приложению A.

#### Фреймворк "Spring for Delphi" (Spring4D)

Платформа Delphi Spring Framework представляет собой обширную коллекцию библиотек, включая коллекции структур данных, контейнер для внедрения зависимостей, перехват, шифрование и другие функции. В этой книге мы подробно рассмотрим библиотеку interception.

Домашняя страница: http://www.spring4d.org/ [3]
Местоположение источника: https://bitbucket.org/sglienke/spring4d [4] 
Тип репозитория: Git 
Лицензия: The MIT License [5]

#### DuckDuckDelphi

DuckDuckDelphi - это фреймворк с открытым исходным кодом, который предоставляет  "duck typing" (утиная типизация) разработчикам Delphi.

Домашняя страница: http://arcana.sivv.com/duckduckdelphi [6]
Местоположение источника: https://code.google.com/p/duckduckdelphi/ [7] 
Тип репозитория: Mercurial 
Лицензия: Apache License V2.0 [8]
